# P6-T02 - Entity/Model

Task: Tạo entity/model cho dự án Restaurant Management System.

Phụ trách: Member 3 + Member 2

Output: Entity classes mapping đúng database schema.

Ghi chú:
- Các entity này dùng cho Spring Boot 3.x với `jakarta.persistence`.
- Nếu project của bạn dùng Spring Boot 2.x, đổi `jakarta.persistence.*` thành `javax.persistence.*`.
- Có thể chỉnh lại tên bảng, tên cột, enum theo `schema.sql` thật của nhóm.

Folder chính:
`src/main/java/com/restaurant/entity/`
